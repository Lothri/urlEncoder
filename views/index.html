URL shortener API

This API receives POST requests and saves URLs to abbreviated addresses, such as '.../1, .../2' and so on.

  • Create a server
  • Send POST request to "https://domain/api/shorturl" with Content-Type: x-www-form-urlencoder with body argument addr=someurl
  • An example:
    POST HTTP 1.1 Type-Content: x-www-form-urlencoded . . . addr=https://youtube.com

Great! Now type https://domain/api/shorturl/1 on the browser. It should redirect you to Youtube.
